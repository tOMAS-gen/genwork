# Data Model: Fix cursor de texto desplazado al editar tareas

No aplica. Esta feature es una corrección visual de CSS sobre un componente de UI existente (calibración de padding entre el `<textarea>` real y su overlay de resaltado). No introduce, modifica ni elimina entidades de datos, campos, relaciones ni transiciones de estado — el texto de la tarea y su persistencia no cambian (ver spec.md > Key Entities, y research.md > Decisión: alcance del fix).
